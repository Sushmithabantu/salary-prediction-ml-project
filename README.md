# Salary Prediction Using Machine Learning

## Project Overview
This project predicts employee salaries based on years of experience using Machine Learning algorithms.

The project applies:
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

The models were trained and evaluated using accuracy metrics and visualizations.

---

## Dataset
Dataset used:
Salary Data Dataset from Kaggle

Features:
- YearsExperience
- Salary

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Algorithms
1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor

---

## Project Workflow
1. Import libraries
2. Load dataset
3. Data visualization
4. Split training and testing data
5. Train machine learning models
6. Predict salaries
7. Evaluate model accuracy
8. Visualize predictions

---

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R2 Score

---

## Visualizations
- Scatter Plot
- Actual vs Predicted Graph
- Regression Line Graph

---

## Results
The models successfully predicted salaries based on years of experience.

Linear Regression and Random Forest achieved high accuracy.

---

## Conclusion
This project demonstrates the implementation of supervised machine learning for salary prediction and model evaluation.
